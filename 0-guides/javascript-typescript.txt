🟦 TypeScript Installation & Configuration Guide
Choose the method that fits your project needs: the quick manual way or the organized compiler way.

🛠️ Method 1: Manual Compilation (Quick Start)
Best for small experiments or single-file testing.

Global Installation: Open your terminal and install TypeScript.

📦 npm i -g typescript

✅ Check version: tsc -v

Create Your File: Create a TypeScript file in your project folder.

📄 touch index.ts

✍️ Write inside: console.log('Typescript work')

Compile Manually: Convert the TS file into a JS file that browsers understand.

⚙️ tsc index.ts (This generates an index.js file)

Connect to HTML:

🌐 touch index.html

🔗 Add this line: <script src="./index.js" type="module"></script>

🚀 Method 2: Compiler Mode (tsconfig.json)
Best for professional projects. It keeps your code organized by separating source files from build files.

Initialize Project: Ensure TypeScript is installed and create your configuration file.

📦 npm i -g typescript

⚙️ tsc --init (This creates tsconfig.json)

Configure: Open tsconfig.json and update the settings to match this structure:

JSON
{
  "compilerOptions": {
    "rootDir": "./",          // Where your source TS files live
    "outDir": "./dist",       // Where the compiled JS files will go
    "module": "nodenext",     // Native ESM/CJS resolution for Node.js
    "target": "esnext",       // Compile to the latest JS version (ES2024+)
    "sourceMap": true,        // Generates .js.map for easier debugging
    "strict": true,           // Enables strict type checking
    "moduleDetection": "auto",// Treats every file as a module automatically
    "skipLibCheck": true      // Speeds up compilation by skipping .d.ts checks
  }
}
Compile All: Run the compiler based on your settings.

⚡ tsc (A dist folder will be created containing index.js)

Connect to HTML:

🌐 touch index.html

🔗 Note: Since the JS is now in the dist folder, use this path:

<script src="./dist/index.js" type="module"></script>

💡 Pro Tip: Watch Mode
Don't want to run tsc every time you save? Use "Watch Mode" to compile automatically whenever a file changes:

👀 tsc -w
